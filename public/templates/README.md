# Templates de Preview

Este diretório contém as imagens e vídeos de preview para os templates de geração.

## Estrutura

### `/scenarios/`
Imagens de preview para os cenários de imagens aprimoradas:
- `table.jpg` - Cenário de mesa profissional
- `nature.jpg` - Cenário de natureza
- `minimal.jpg` - Cenário minimalista
- `lifestyle.jpg` - Cenário lifestyle
- `studio.jpg` - Cenário de estúdio
- `random.jpg` - Cenário aleatório

**Dimensões recomendadas:** 400x400px (aspect ratio 1:1)

### `/videos/`
Thumbnails e vídeos de preview para os templates de vídeo:

#### Modelo 1 - Dinâmico e moderno
- `template-1-thumb.jpg` - Thumbnail
- `template-1-preview.mp4` - Preview do template (5-10s)

#### Modelo 2 - Elegante e minimalista
- `template-2-thumb.jpg` - Thumbnail
- `template-2-preview.mp4` - Preview do template (5-10s)

#### Modelo 3 - Energético e vibrante
- `template-3-thumb.jpg` - Thumbnail
- `template-3-preview.mp4` - Preview do template (5-10s)

#### Modelo 4 - Profissional e clean
- `template-4-thumb.jpg` - Thumbnail
- `template-4-preview.mp4` - Preview do template (5-10s)

**Dimensões recomendadas:** 
- Thumbnails: 640x360px (aspect ratio 16:9)
- Vídeos: 1280x720px ou 1920x1080px, duração 5-10 segundos

## Notas

- As imagens devem ser otimizadas para web (formato WebP recomendado)
- Os vídeos devem ser comprimidos e otimizados (H.264, MP4)
- Substitua os placeholders por arquivos reais para melhor experiência
